# Rematch Physical Therapy

Marketing website for Rematch PT — online physical therapy by Dr. Allie.

## Pages

- `index.html` — Homepage
- `injuries.html` — Injuries & services
- `methodologies.html` — Process & comparison
- `about.html` — About Dr. Allie
- `contact.html` — Free consultation form

## Stack

Pure HTML, CSS, and vanilla JavaScript. No frameworks, no build step, no dependencies.

## Deploy

Drop these files on any static host — Netlify, Vercel, GitHub Pages, Cloudflare Pages, or a simple web server.

### GitHub Pages

1. Push to a repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your site will be live at `https://yourusername.github.io/rematch-pt/`

### Netlify

1. Drag the project folder into [Netlify Drop](https://app.netlify.com/drop)
2. Done — you'll get a live URL instantly
